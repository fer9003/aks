## AKS UI Azure Portal

1. Una vez desplegado el cluster AKS desde Azure Portal, Ingresar a la shell de azure desde el navegador
https://shell.azure.com

2. Descargar el archivo config para poder administrar el cluster
`az aks get-credentials --resource-group <name_of_resource_group> --name <name_cluster>`

3. Listar info del node
`kubectl get nodes -o wide`

4. Listar los namespaces
`kubectl get namespaces`

5. Listar los Pods de todos los namespaces
`kubectl get pods --all-namespaces`
